ICT Accessibility Requirements Wizard

The ICT Accessibility Requirements Wizard is a preliminary tool that must be completed to ascertain if accessibility is required for your project. The goal of this tool is to help identify potential accessibility requirements; and acts as a point of first contact for managers and the IT Accessibility Office (ITAO).

Instructions

To best utilize this tool, follow each step in order and consult the Supporting Information section for more information on accessibility. Please contact the ICT Accessibility Strategy generic mailbox if you need help when using the tool.

  1. Fill out the ICT Accessibility Requirements Wizard. If any choices apply to your project, continue to the next steps.
  2. Complete to the best of your knowledge Step 1: Define ICT functionality.
  3. Go to Step 3: Generate requirements document and click on Full requirements table with informative sections (English) and save document.
  4. Send the ICT Accessibility Requirements document to the ICT Accessibility Strategy generic mailbox.
